There are several ways to contribute to the ongoing effort that is Almonaster:

  • Contributions to the game itself. Almonaster is perennially under development, and help is always welcome. Such help might include bug reports; code patches; ideas and suggestions; new theme artwork, alien icons, button sets or planet graphics; corrections to the existing documentation or new articles; or any other improvements to the existing game.

  • Contributions to the servers. Money to support dedicated hosting facilities would provide a better in-game experience for all users.

  • Contributions to the administrator. Running a public game server with no membership fees, sponsors or in-game advertisements can be expensive and time-consuming. While there is no obligation whatsoever, if you feel that the time you spend on the server is worth a contribution, you can donate using Paypal. Small amounts are fine, as are credit card payments. New signups for Paypal can use this link as a referral.

Although $1 per month might seem insignificant, these micro-payments do add up and are greatly appreciated.

    Financial contributions are collected and placed in a general fund that is used for server expenses, which are either hardware or internet connection costs. These expenses generally outweigh the volume of contributions. Currently there are no explicit rewards for contributors except for the administrator's gratitude and the promise to keep the server alive. New ideas for how to give back to specific contributors are always welcome.
