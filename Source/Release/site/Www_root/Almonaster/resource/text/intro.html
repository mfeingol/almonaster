Almonaster is a turn-based multi-player space war game that is free for all players. The game is fully web-based. If you are reading this page, you can play.

Almonaster was created by Max Feingold (design, programming and user interface) and Jens Klavsen (graphics, icons, user interface). Bug reports, suggestions and criticism are welcome. Almonaster source code is freely available and is licensed under the GPL. Like most free software, Almonaster comes with no warranty.

Almonaster is a variant of the online game Stellar Crisis.

If you are new to the game, enter the empire name and the password that you would like to use while playing and press the login button. Once you have logged in, you can read the documentation for information on strategies and gameplay.
