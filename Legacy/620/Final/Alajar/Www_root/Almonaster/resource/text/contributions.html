There are several ways to contribute to the ongoing effort that is Almonaster:

  • Contributions to the game itself. Almonaster is in constant development, and help is always welcome. This includes bug reports, code patches, ideas, suggestions, artwork, new alien icons, button sets, planet graphics, corrections to the documentation, new articles, and any other improvements to the existing game.

  • Contributions to the servers. The machine hosting this server is a 400 MHz K6-III with 256MB of RAM. Faster hardware would provide a better game experience for all users.

  • Contributions to the administrator. Running a public game server with no membership fees, sponsors or advertisements can be an expensive undertaking. Users who feel that the time they spend on the server is worth a contribution can use the free Paypal payment service. Small amounts are fine, and there is no obligation whatsoever, except for one's own generosity. New signups for Paypal through this link will donate $5 as a referral reward.

    Donate a fixed amount, or subscribe:

    $5 per month  $6 per month  $7 per month  $8 per month 

    Paypal charges 2.9% plus 30 cents on every transaction. To compensate for this, the amounts charged for monthly subscriptions will be slightly higher than the amounts displayed above. Although $1 per month might seem insignificant, these micro-payments do add up and are greatly appreciated.

    Financial contributions are collected and placed in a general fund that is used for server expenses, which are either hardware or internet connection costs. These expenses generally outweigh the volume of contributions, but the connection is also used for other purposes and would be paid for regardless of the server. Currently there are no explicit rewards for contributors except the administrator's gratitude and the promise to keep the server alive, although ideas related to how to give back to contributors are always welcome.

For more information or for suggestions about contributions, please send email to almonaster@myrealbox.com.
