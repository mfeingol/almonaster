Almonaster is a turn-based multi-player war game that is free for all players. The game is fully web-based and requires no downloads, Javascript, Java or ActiveX controls. If you are reading this page and your browser can post forms, then you can play.

Almonaster was created by Max Attar Feingold (design, programming and user interface) and Jens Klavsen (graphics, icons, user interface). Bug reports, suggestions and criticism are welcome.

More information about Almonaster can be found at the official site, the mailing list and the project newsgroups, all hosted by SourceForce.

Almonaster is licensed under the GPL. Server source code is freely available. Like most free software, Almonaster comes with no warranty. Its authors take no responsibility for its misuse or the damage that it might cause.

Almonaster shares many similarities with Stellar Crisis. Resources that may be useful for a better understanding of the culture and gameplay of both games include:

  • Al Camp's Stellar Crisis Room.
  • The Yahoo! Stellar Crisis Club, where all important SC discussion takes place.
  • The SC-Talk mailing list, graciously hosted by Nathan Mehl.
  • The Stellar Crisis FAQ, maintained by Al Camp.
  • Erick Beck's Faces of Stellar Crisis and Guide to Stellar Crisis Etiquette.

If you are new to the game, enter the empire name and the password that you would like to use while playing and press the login button. Once you have logged in, you can read the documentation for information on strategies and gameplay.
